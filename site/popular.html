    Como obter um Client ID?
    1. Acesse o Spotify Developer Dashboard e faça login.
    2. Clique em Create app. Nomeie como desejar (ex: "Angra Lyrics").
    3. No campo Redirect URIs, adicione exatamente o URL atual desta página (ex: ). Marque a opção "Web API".
    4. Salve, copie o Client ID e cole no campo acima.
